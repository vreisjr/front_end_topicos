Introdução ao HTML

  1. A anatomia das tags:
    • O elemento principal do HTML é a Tag
    • Gearlmente, as tags tem abertura e fechamento
    • Exemplo: A tag p é de parágrafo

  2. A estrutura do HTML
    • A estrutura base do HTML é composta por:
    • DOCTYPE - Tag que declara a versão do HTML
    • html - Tag que envolve todo o código HTML
    • head - A Tag que colocamos todas as configurações de um site, como, a importação do CSS
    • body - É onde todos os elementos visíveis estão

  3. Títulos
    • São conecidos como "Headings"
    • Utilizado para separar seções
    • O nome da Tag é a h, onde o valor pode ser de 1 a 6
    • O maior e mais importante é o h1, clocamos um por página
    • A importância deve estar conectada com o propósito da página
    • Esse assunto faz parte do HTML Semântico, um tópico mais avançado

  4. Parágrafos
    • Utilizamos Parágrafos para textos maiores
    • A Tag é a p
    • Cada Parágrafo começa uma nova linha, e este comportamento acontece com todas as Tags de bloco
    • Existem várias tags de bloco, os Títulos também fazem parte

  5. Tgas sem conteúdo
    • Tags sem conteúdo possuem recursos geralmente como quebrar linha
    • para este fim, podemos utilizar a Tag br
    • Para uma linha horizontal, temos hr
    • Elas também introduzem o conceito de Self Closing Tags, onde a Tag não possui outra Tag e fechamento

  6. Comentários no HTML
    • Comentários são utilizados para descrever como algo funciona no código
    • Ou explicar para outros programadores o que fizemos
    • Os comentários não são exibidos na página
    • Mas qualquer um que inspecionar o seu código, terá acesso aos comentários
    • Atalho: ctrl + ;

  7. Atributos
    • Atributos podem ser utilizados para adicionar funcionalidades as Tags
    • A Tag a é responsável por direcionar a uma nova página ou site
    • Mas aonde vamos adicionar o endereço/URL?
    • Colocamos um atributo chamado href

    • Exemplo: a href="https://www.google.com"

      • Site do Google

  8. Abrir Link em uma nova aba
    • Com um atributo podemos fazer o link abrir em uma nova aba
    • Isso é utilizado frequentemente para redirecionar a outro site
    • Por exemplo: temos um link que leva a um e-commerce que não é o nosso, não temos quele domínio
    • Então utilizamos o atributo target com o valor_blank

    • Exemplo: a href="https://www.youtube.com/" target="_blank"

      • Abrir Youtube em uma nova aba

  9. Imagem no HTML
    • Nós podemos inserir imagens no nosso site com a Tag img
    • O caminho relativo até a imagem é inserido no atributo src
    • Normalmente, colocamos as imagens em uma pasta chamada: img ou assets, para fins de organização
    • Nota: a img é uma Self Closing Tag

    • Exemplo: img src="./src/img/city.jpg"


  10. Atributo alt
    • Nas tags de img, temos um atributo chamado alt
    • Inserimos um texto nele para descrever a imagem
    • Este recurso é impostante para acessibilidade, fzendo com que nosso site seja melhor rankeado pelo Google também

    • Exemplo: img src="./src/img/city.jpg" alt="Imagem de uma cidade a noite"

  11. Listas não Ordenadas
    • Listas são importantes para muitos sites
    • Podemos crira até um menu a partir de uma lista
    • As não ordenadas são criadas pela Tag ul
    • Cada item da lista é representado pela Tag li

    • Exmplo:

      • Maçã
      • Pera
      • Uva
      • Banana

  12. Listas Ordenadas
    • São interessnates para quando há um procedimento, ou passos a serem seguidos
    • Utilizamos a Tag ol
    • E os itens continuam sendo a Tag li

    • Exemplo: Agenda de rotina

      1. Acordar
      2. Tomar café
      3. Ir trabalhar
      4. Intervalo de almoço
      5. Voltar ao trabalho
      6. Ir pra casa
      7. Jantar
      8. Dormir

  13. Tabelas
    • Hoje é considerada obsoleta
    • Ussmos tabelas para exibir dados que podem ser categorizados em colunas
    • Tabelas são estruturas complexas no HTML, e não tão utilizadas
    • A Tag é a table, isso cria uma tabela
    • E também um cabeçalho e um corpo
    • Cada linha é criada com a Tag tr, e os dados em td
    • No cabeçalho utilizamos a Tag th

    • Exemplo:

      • ID: Nome: Profissão:
      1 Matheus Programador
      2 Alexia Engenheira civil

  14. A Tag div
    • A Tag div é utilizada para criar divisões/seções no nosso site
    • Podemos criar elementos menores também, como cards
    • O principal propósito é: encapsular elementos que estão conectados entre si

    • Exemplo:

      • Título da seção

      Descrição da seção, um texto relacionado a este assunto
